# 🤖 AI Data Talk - Chat with Your Data using Groq LLM

AI Data Talk (GRIA) is an intelligent data analysis application that allows users to interact with CSV datasets using natural language. Powered by the Groq API and Large Language Models (LLMs), the application provides instant insights, data summaries, statistical analysis, and visualization recommendations without requiring SQL or programming knowledge.

## 🚀 Features

- 📊 Upload CSV datasets
- 💬 Chat with your data using natural language
- 🤖 Powered by Groq LLM
- 📈 Automatic data summaries
- 🔍 Statistical analysis
- 📉 Generate insights from datasets
- 🧹 Data preprocessing
- ⚡ Fast inference using Groq API
- 🔐 Secure API key management using `.env`
